- **Added conditional logic for grid entities with override** - More flexible grid entity behavior
- **Added People module** - Show person status, location, avatar, and configurable data
- **Added default title and value size in graph module** - Customize graph text sizing

- **Added Navigation module with dock-style layout** - Global navigation bar with icon paths, routes, media widget, and multiple style presets
- **Added time interval toggle for graphs** - Choose time ranges for history and forecast
- **Added show/hide dock toggle when editing** - Dock can be hidden so it does not cover the save button
- **Added navigation module scrolling on mobile** - Easier use on small screens
- **Added attribute display in icon and info modules** - Show entity attributes in addition to state
- **Added template cheatsheet button with copy support** - Quick reference for templates next to Unified Template fields

- **Improved graph module value refresh** - Graphs update more reliably
